# Calculadora Java con Swing

Esta es una calculadora básica desarrollada en Java utilizando Swing.

## Instrucciones

### Compilación
Puedes compilar el código fuente utilizando un IDE como Eclipse o NetBeans, o mediante la línea de comandos con `javac`. Asegúrate de tener instalado el JDK de Java.

### Ejecución
Para ejecutar la aplicación, puedes iniciar la clase principal `NewJFrame`. Esto se puede hacer desde un IDE o desde la línea de comandos.

```bash
java NewJFrame
